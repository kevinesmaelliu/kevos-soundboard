# Soundboard Audio Files

Place your audio files in this folder. Supported formats:
- .mp3
- .wav  
- .ogg
- .m4a

The soundboard will automatically load all audio files from this directory.

## Example files to add:
- airhorn.mp3
- drumroll.wav
- applause.mp3
- crickets.wav
- sad-trombone.mp3
- wow.wav
- nope.mp3
- success.wav
